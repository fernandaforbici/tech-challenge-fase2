# Tech Challenge - Fase 2  
## API de Blog Educacional

Este projeto consiste no desenvolvimento de uma API RESTful para gerenciamento de postagens de um blog educacional, permitindo que docentes criem conteúdos e alunos possam acessá-los.

---
# Tecnologias Utilizadas

- Node.js
- Express.js
- PostgreSQL
- Docker & Docker Compose
- Swagger (Documentação)
- Jest + Supertest (Testes)
- GitHub Actions (CI/CD)
